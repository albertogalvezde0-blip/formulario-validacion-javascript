### Descripción / Description
Este proyecto consiste en un formulario de registro interactivo desarrollado como parte de una tarea académica. El objetivo principal fue implementar validaciones en el lado del cliente utilizando JavaScript nativo, asegurando que los datos introducidos cumplan con ciertos criterios antes de ser enviados.

Además de la funcionalidad, le he dado un toque visual especial añadiendo un fondo animado con partículas interactivas que reaccionan al movimiento del ratón, creado con HTML5 Canvas.

This project is an interactive registration form developed as part of an academic assignment. The main goal was to implement client-side validation using vanilla JavaScript, ensuring that input data meets specific criteria before submission.

Beyond functionality, I added a unique visual touch with an animated background featuring interactive particles that react to mouse movement, built using HTML5 Canvas.

### Características / Features
- **Validación en tiempo real / Real-time Validation**: Comprobación de email, complejidad de contraseña, longitud de usuario y nombre completo.
- **Interfaz Interactiva / Interactive Interface**: Feedback visual inmediato (bordes rojos para errores) y mensajes de alerta claros.
- **Diseño Visual / Visual Design**: Estilos personalizados en CSS y un fondo dinámico de partículas conectadas.

### Cómo usarlo / How to use
Simplemente abre el archivo `index.html` en tu navegador web favorito o visita el enlace de la demo. No se requiere instalación ni dependencias externas.

Simply open the `index.html` file in your favorite web browser or visit the demo link. No installation or external dependencies are required.
