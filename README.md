### Description
This project is an interactive registration form developed as part of an academic assignment. The main goal was to implement client-side validation using vanilla JavaScript, ensuring that input data meets specific criteria before submission.

Beyond functionality, I added a unique visual touch with an animated background featuring interactive particles that react to mouse movement, built using HTML5 Canvas.

### Features
- **Real-time Validation**: Checks for email format, password complexity, username length, and full name.
- **Interactive Interface**: Immediate visual feedback (red borders for errors) and clear alert messages.
- **Visual Design**: Custom CSS styling and a dynamic connected-particles background.

### How to use
Simply open the `index.html` file in your favorite web browser or visit the demo link. No installation or external dependencies are required.
